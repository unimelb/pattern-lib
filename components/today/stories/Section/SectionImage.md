# Section Image component

The component `SectionImage` acceps the following props:

| props    	| type    	|
|----------	|---------	|
| tagName   | String  	|
| img      	| Object  	|

Usage Example:

```html
<SectionImage tagName="div" :img="{ src: 'https://via.placeholder.com/800x800', alt: 'test alt' }"> Section Content </SectionImage>
```
