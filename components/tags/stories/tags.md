## Tags component

The component props is:

- tags
  > An array of href and value eg: 
```json 
  [{
    href: 'https://www.unimelb.edu.au/',
    value: 'tag1',
  },
  {
    href: 'https://www.unimelb.edu.au/',
    value: 'tag2',
  }]
```

### Usage example
```html
<tags data="[{
    href: 'https://www.unimelb.edu.au/',
    value: 'tag1',
  },
  {
    href: 'https://www.unimelb.edu.au/',
    value: 'tag2',
  }]"></tags>
```
