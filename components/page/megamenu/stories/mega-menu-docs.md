## Mega Menu

The Mega Menu componet receive an Array called `items` which build the Menu itself

Example:

```
:items='[
  {
    "title": "Menu title",
    "href": "/url",
    "items": [
      {
        "title": "Sub-menu title",
        "href": "/sub-url",
      },
      {
        "title": "Sub-menu title 2",
        "href": "/sub-url-2",
      },
    ]
  },
  {
    "title": "Menu title 2",
    "href": "/url-2",
  }
]'
```
